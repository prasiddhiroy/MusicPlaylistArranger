# Music Playlist Arranger (C Project)

This project allows users to manage a simple music playlist.

### Features:
- Add Song  
- View Song List  
- Search Song  
- Delete Song  
- Save playlist to file  
- Load playlist at startup  

### Folder Structure:
- src/ → All .c files  
- include/ → All .h files  
- docs/ → Project documentation  
- data/ → Data storage (songs.txt)
